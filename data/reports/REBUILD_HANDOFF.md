## Environment notes
- `export PYTHONPATH=src` before running anything.
- Python is `.venv/bin/python` (macOS, no `timeout` CLI — use Python-side timeouts).
- **Network works** in this environment: nba_api (`playerindex`, `leaguedashptstats`,
  `leaguedashplayerstats/teamstats`) and Basketball-Reference all return real data.
  Raw responses cache under `data/raw/nba_api`, so re-runs are fast/offline.
- Processed/report `*.parquet` are **gitignored by design** — they are regenerated by
  pipeline scripts. Commits contain code + Markdown/CSV reports, not binary data.
